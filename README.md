# Commonwealth Bank Data Engineering Virtual Experience

This repository contains the projects I completed as part of the Commonwealth Bank Data Engineering Virtual Experience Program.

## Tasks Completed

### Task 1 — Supermarket Transaction Analysis
- Analyzed supermarket transaction data using Python and Pandas
- Calculated customer purchase insights and spending patterns

### Task 2 — Customer Data Anonymisation
- Applied data masking and anonymisation techniques
- Protected sensitive customer information while preserving analytical value

### Task 3 — Twitter Data Analysis Proposal
- Proposed business insights using Twitter analytics
- Suggested sentiment analysis and engagement analysis approaches

### Task 4 — Twitter Database Design
- Designed a relational database structure for storing Twitter data
- Defined tables, primary keys, and relationships
